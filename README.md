## IDEs

### Visual Studio Code

- Install the extension "PlantUML" of jebbs (jebbs.plantuml)
- Open the view "PlantUML Preview Current Diagram"

### Eclipse

- Install the  plugin "PlantUML" of the Norwegian Univ. of Science and Tech
- Open the view "PlantUML"
